# RESEARCH-PAPER
MediQueue+ is an AI-powered healthcare platform that analyzes patient symptoms using DistilBERT and recommends the right doctor. It includes real-time queue management, emergency SOS alerts, and digital prescriptions to improve healthcare accessibility, especially in rural areas.

MediQueue+ is an AI-powered healthcare management system designed to solve rural healthcare challenges by providing intelligent doctor recommendations and real-time patient management.

The system uses a fine-tuned DistilBERT model to analyze patient-described symptoms and recommend the most suitable medical specialist with high accuracy.

It also integrates a complete healthcare workflow including:

AI-based symptom analysis & doctor recommendation
Token-based queue management system
Real-time emergency SOS alert system
Digital prescription generation
Secure doctor dashboard with live patient monitoring

Built using Flask, SQLite, and Socket.IO, this platform ensures efficient, fast, and accessible healthcare services, especially for underserved and rural areas.
